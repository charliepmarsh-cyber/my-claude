## WEEK-BY-WEEK LAUNCH PLAN

### Week 1: Deploy Your Demo + Set Up Upwork

**Day 1-2: Deploy live demo**
- Deploy the frontend to Vercel/Netlify (free tier, 5 minutes)
- URL becomes: pulsecommerce-demo.vercel.app
- This is your sales weapon - every proposal links to it

**Day 3: Upwork profile setup**
- Title: "I'll Build You an AI-Powered E-Commerce Analytics Agent with Multi-Channel Attribution"
- Profile description: Use the portfolio description from the Upwork listing PDF
- Portfolio item: Screenshot of dashboard + link to live demo
- Skills tags: shopify, analytics, n8n, ai, react, dashboard, attribution
- Rate: GBP75-100/hr (signals quality, but you sell fixed-price)

**Day 4-5: First 10 proposals**
- Search for: "shopify analytics", "ecommerce dashboard", "shopify reporting", "attribution tracking", "ecommerce data"
- Filter: Budget GBP1,000+, posted in last 7 days
- Send 10 proposals using the Fast Pitch template (50-80 words)
- ALWAYS include: "I have a live demo I can show you in 15 minutes"

**Day 6-7: Follow-up + more proposals**
- Send 10 more proposals
- Follow up on any views/shortlists from Day 4-5
- Start building Upwork presence (answer questions in e-commerce forums)

### Week 2: First Conversations

**Target:** 3-5 replies from 20 proposals (15-25% response rate is normal)

**For each reply:**
1. Send the demo link immediately
2. Book a 15-min call using Calendly (free tier)
3. On the call, use the Discovery Script from the business plan
4. After call, send fixed-price proposal on Upwork within 2 hours

**Proposal format on Upwork:**
```
Hi [Name],

Great call today. Based on what we discussed, here's what I recommend:

PulseCommerce [Growth/Enterprise] Build
- [List specific features they need]
- [Mention their channels]
- Deployed in [X] days
- Includes [X] months of maintenance

Investment: GBP[X] one-time + GBP[X]/mo maintenance

I'll start as soon as you accept. First milestone: your Shopify data 
flowing in a branded dashboard within 48 hours.

[Your name]
```

### Week 3-4: Close First Client + Deliver

**Target:** 1 client at GBP1,800-4,500

**Delivery priority order:**
1. Get Shopify data flowing (Day 1) - client sees real numbers = trust
2. Apply branding (Day 2) - feels like "theirs"
3. Build remaining pages (Days 3-7)
4. Demo call with real data (Day 8)
5. Go live (Day 10-14)

**While delivering:**
- Continue sending 5 proposals/week
- Pipeline never stops

### Month 2-3: Scale to 3-5 Active Projects

**Weekly rhythm:**
- Monday: Send 5-10 new proposals
- Tuesday-Thursday: Client delivery work
- Friday: Follow-ups, retainer tasks, demo improvements

**Revenue target Month 2:** 2 new clients = GBP3,600-9,000
**Revenue target Month 3:** 3 new clients = GBP5,400-13,500

## PROPOSAL OPTIMIZATION TIPS

**What gets you shortlisted:**
1. Mention their specific store/brand name
2. Reference the channels THEY mentioned
3. Include "I have a live demo" (nobody else does)
4. Fixed price, not hourly
5. Specific delivery timeline ("7 days" not "1-2 weeks")
6. One line about AI referral tracking (unique hook)

**What kills proposals:**
1. Generic "I can build you a dashboard"
2. Hourly rate focus
3. No demo or portfolio link
4. Long paragraphs about your experience
5. Asking questions before showing value
6. "Let me know if you're interested" (weak CTA)

**Strong CTAs (pick one per proposal):**
- "Want me to send the demo link?"
- "I can show you this working in 15 minutes - when works?"
- "Reply with your Shopify URL and I'll put together a custom scope"
